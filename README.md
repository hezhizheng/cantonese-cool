### Why ？
> 广东话要学一世，无奈口音太难改，特别是碰到  `政、静` 、`攰、贵` `上、想` 这样发音差不多的字眼更是不知道讲的到底是哪一个字。
> 然后就在网上找了一些能讲广东话的网站，发现用是能用，就是不太方便，就想着要不自己搞一个？顺便练手一下小程序（一直想试着写一个却始终不知道写个啥），
> 然后就有了这个小程序了。


### Features
- 支持输入的文本通过音频以广东话的形式播放
- 支持粤拼显示
- 支持收藏播放过的文本

### 使用
- 基于uniapp、HBuilder X 开发
